# OMaa Planner iOS App - Legal Pages

This repository hosts the legal and policy pages for the OMaa Planner iOS application.

## Pages

| Page | Path | Description |
|------|------|-------------|
| Privacy Policy | `/ios-privacy-policy` | Data collection, storage, and third-party services |
| Terms of Service | `/ios-terms-of-service` | Subscription terms, auto-renewal, and cancellation |
| Return Policy | `/ios-return-policy` | Refund process through Apple |

## Live URLs

Once deployed to GitHub Pages, these pages will be available at:

- https://o-maa.github.io/ios-privacy-policy
- https://o-maa.github.io/ios-terms-of-service
- https://o-maa.github.io/ios-return-policy

## Setup

1. Enable GitHub Pages in repository settings
2. Set source to deploy from the main branch
3. Pages will be served at `https://o-maa.github.io/`
